# 文件格式转换

200+ 文件格式相互转换。

使用方法：将文件（支持多个）拖入 uTools 输入框，选择 “文件格式转换”，等待文件上传完成，然后选择目标格式，开始转换，转换完成后点击 “下载” 按钮下载。

注：

1. 文件大小限制为 10M
2. 转换后的文件只能下载一次
3. 不要同时上传太多文件，否则可能会导致网络错误转换失败
